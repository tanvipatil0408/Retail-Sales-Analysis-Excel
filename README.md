# Retail Sales Analysis Using Excel

## Project Objective
Analyze retail sales data to identify sales trends, regional performance, profitability, and top-performing products using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs

## Analysis Performed
- Total Sales by Region
- Total Profit by Category
- Monthly Sales Trend Analysis
- Top Sub-Categories by Sales

## Key Insights
- West region generated the highest sales.
- Technology category showed strong profitability.
- Monthly sales trends highlighted seasonal performance changes.
- Certain sub-categories contributed significantly to total sales.
